Summary

  • <meta charset="UTF-8"> → ensures all characters display correctly.
  • <audio> → plays music, podcasts, sound effects.
  • <video> → plays movies, tutorials, with controls & subtitles.
  • <iframe> → embeds external content like videos, maps, docs.
