Notes & Best Practices

  • Put the nested <ul> or <ol> INSIDE the parent <li>.
  • Avoid extremely deep nesting — it becomes hard to read and confusing for users and screen readers.
  • You can change marker styles with CSS (e.g., list-style-type), but keep default markers for clarity in beginner lessons.
